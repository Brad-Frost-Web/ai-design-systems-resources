The third leg of the asset stool — the set of instructions that lets both makers and users wield the system's materials; without it, the system falls down (the course's 'shelving unit without instructions' metaphor). It lives inline in Figma, as code comments surfaced through Storybook auto-docs, and, because those tend to silo, on consolidated reference sites. The course notes getting documentation right matters even more in the AI era because it provides context to the models.
