An agentic AI coding tool that lets you interact with the files on your computer and connect to other things via MCPs. In the demo it's run inside the Warp terminal and connected to Figma through Figma Console MCP to generate and build a website. The course lists it (alongside Cursor and Windsurf) among the agentic tools that went mainstream in developer workflows by 2025.
