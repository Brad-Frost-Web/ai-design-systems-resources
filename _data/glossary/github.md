GitHub is a service/platform (with "Git" right in the name) that hosts Git-managed code repositories in the cloud, making them the source of truth for a project's codebase where all code, whether authored by humans or AI agents, ultimately lands. Beyond just hosting, it provides tools to share, review, manage, and collaborate on code, and the course calls it the industry-standard place to publish open work. Similar platforms include GitLab, Bitbucket, Azure Repos, and Codeberg.
