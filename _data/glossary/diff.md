A diff is how Git surfaces changes by paying attention to lines of code and pulling them up side by side. If you delete a line it shows bright red, and if you add a line it shows up green. The course calls going through code line by line — asking "did this change, yes or no?" — the heart and soul of working with version-controlled code.
