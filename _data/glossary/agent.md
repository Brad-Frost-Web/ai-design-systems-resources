An AI that can take action — not just respond, but do things. The course's signature metaphor is 'an LLM with hands': the agent understands what the language model's responses are asking for and carries out the task because it can interact with the world. Agents can read files, run code, search the web, and call APIs. The canonical example is Claude Code, described as one of the first agent platforms, which can edit your codebase, run tests, commit changes, interact with APIs, CLIs, and MCPs, and has full awareness of the system it's installed on; Cursor's built-in agents are another example.
