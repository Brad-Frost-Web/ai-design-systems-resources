Git is free, open-source version control software for managing code projects that has been around for about twenty years and revolutionized how developers collaborate. It is distributed, meaning you install it on your own machine where it works offline inside your codebases, keeping track of changes over time. The course frames Git as especially important in the AI age: because AI can make a gigantic mess in a codebase, Git gives you a safe place to explore, refine, roll things back, and merge code so that if things go sideways no harm is done.
