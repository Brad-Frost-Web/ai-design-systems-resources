The example design system used throughout the course's hands-on demonstrations. Its component library lives in Figma and is built in code as Lit web components with SCSS styling, Storybook stories, and Playwright tests, organized in a monorepo (e.g., a Soul Patrol web components package). Components carry an slpl- prefix (e.g., slpl-text-field) and use Soul Patrol design tokens, and the course uses it to demonstrate FigmaLint, the Figma MCP, the Figma Console MCP, and Story UI.
