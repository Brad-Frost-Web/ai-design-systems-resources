The common building blocks an organization needs to create user experiences — branded, reusable controls (like buttons) that capture UX, visual, brand, accessibility, and front-end best practices so makers don't rebuild them from scratch. They live across design tools, code, and documentation; using them, designers can drag and drop and developers get the best practices 'for free.'
