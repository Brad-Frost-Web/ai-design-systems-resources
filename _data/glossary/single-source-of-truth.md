The course describes a Git-managed repository hosted in the cloud on a service like GitHub as the source of truth for a project's codebase — the place where all code, whether authored by humans, AI agents, or some combination, ultimately finds its way. Because it lives off your own machine, it also serves as a backup you could clone down to a new computer if yours were lost.
