Introduced (per the course, by Brad's friend Jina) to describe design primitives — the foundational values like color — that can be fed out and used across all of an organization's design libraries, code libraries, and documentation, including native mobile. In code they are written in platform-agnostic JSON and transformed (via Style Dictionary) into formats like web CSS custom properties, iOS, or Android. The course organizes them into tiers (tier one, tier two, tier three).
