A collection of reusable UI components used to create and power user interfaces. The course traces its evolution from copy-pasteable UI kits and libraries like Bootstrap, through React component libraries made into real dependencies via ES modules, to today's design and code component libraries that should stay aligned across Figma and Storybook.
