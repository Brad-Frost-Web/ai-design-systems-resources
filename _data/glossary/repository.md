A repository is your project: the folder that contains all the source files your software needs to run, where Git is installed and which ultimately gets pushed up and hosted on a service like GitHub. The course also describes it more technically as just a directory with a hidden .git subfolder inside it. Once published to GitHub, the remote repository acts as a backup and source of truth living off your own machine.
