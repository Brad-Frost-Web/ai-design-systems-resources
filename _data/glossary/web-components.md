A native web technology, introduced in the course's history, that made design systems much easier to deliver across many tech stacks. Because web components are compatible with any website regardless of what it's built in, they let teams deliver presentational components — the look, feel, and structure — to any web experience instead of maintaining separate framework-specific implementations.
