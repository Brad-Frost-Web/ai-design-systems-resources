The tool the course uses to publish a design system's transpiled distribution folder as a package and to let downstream product teams install it. After running a build command to produce the distribution folder, the library is published via NPM; consumers then run an install command (e.g. the Eddie web components or Carbon web components install) to bring it into their project.
