A free, open-source Figma community plugin built by Southleft (TJ) that audits design system components for AI readiness. After a first-run setup where you choose an AI model provider (Claude, ChatGPT, or Gemini) and paste an API key, it runs a component audit that scores AI-readiness and checks for detached instances, hardcoded values, design token usage, layer naming, component descriptions, accessibility, and more. It is not only an analysis tool: it can auto-fix the issues it finds (bulk-renaming layers, mapping hardcoded values to existing variables, and generating component descriptions) so a component can reach a 100% score and be confidently handed off to development or MCP-powered design-to-code workflows.
