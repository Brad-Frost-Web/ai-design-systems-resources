GitHub CLI (installed via the gh command) is a command line tool that lets you interact with GitHub through the terminal. The course calls it a real game changer and a big unlock for AI-powered work, because it lets you use natural language through an agent like Claude Code to say things like "create a new branch," "issue a pull request," or "file an issue," with the agent interfacing with the CLI to actually carry out those actions on GitHub.
