Cloning takes an existing repository hosted on a service like GitHub and copies it down onto your local machine; git clone is the command you run. The course frames it as how you'd pull your design system's repository onto your machine, or recover a project onto a new computer from the GitHub source of truth, and notes GitHub's green "Clone" button provides the commands to paste into your terminal.
